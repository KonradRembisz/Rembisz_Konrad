Detailed Description

Klasa Sequence Klasa Sequence jest definicją obiektu, który przechowuje dwie wartości typu int i udostępnia metody do ich manipulacji i wykonywania na nich prostych działań matematycznych.

◆ dodaj()

float Sequence::dodaj ( )

Wzór na rozkład F-Snedecora.

\[ F = \frac{\left(\frac{S_1}{n_1}\right)}{\left(\frac{S_2}{n_2}\right)} \]

gdzie \(S_1\) i \(S_2\) są odchyleniami standardowymi dla dwóch badanych próbek, a \(n_1\) i \(n_2\) są ich odpowiednimi rozmiarami.

Rozkład F-Snedecora jest używany do testowania hipotez dotyczących równowagi średnich dla dwóch lub więcej grupy.
